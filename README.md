## 如何运行

### 方法一：直接在浏览器中打开
1. 找到 `index.html` 文件
2. 双击该文件，系统会自动用默认浏览器打开
3. 开始游戏

### 方法二：使用桌面浏览器打开
1. 打开任意现代浏览器（如 Chrome、Firefox、Edge 等）
2. 点击浏览器的 "文件" 菜单
3. 选择 "打开文件"
4. 找到并选择 `index.html` 文件
5. 开始游戏

## 游戏操作

1. 点击开始游戏按钮开始游戏
2. 点击棋盘上的交叉点下棋
3. 当形成正方形或直线时，会获得额外的下棋机会
4. 棋盘下满后，可以开始吃子
5. 任意一方可以点击认输按钮提前结束游戏
